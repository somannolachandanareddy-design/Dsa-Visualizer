# 📊 DSA Problem Visualizer

A Python-based tool to visualize sorting algorithms using graphical representations. This project helps in understanding how algorithms work step-by-step.



 Features

* Visualizes Bubble Sort and Merge Sort
* Step-by-step graphical representation
* Helps in learning algorithm behavior



 Tech Stack

* Python
* Matplotlib
